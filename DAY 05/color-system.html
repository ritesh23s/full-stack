Color Systems

  1. Named Color: for example → color:black;

    2. Each browser recognizes around 140 named colors.

    Example: red, blue, pink, purple, etc.

    for css example: 
                h4{
                color: orange;
            }
            
            p{
                color: blue;
            }
  2. RGB Color: for example → color:rgb(0,0,0);

    3. RGB stands for Red Green Blue(0-255)

    red is rgb(255,0,0)

    green is rgb(0,255,0)

    blue is rgb(0,0,255)

    black is rgb(0,0,0)

    yellow is rgb(255,255,0)

    note: Here, "color:rgb(red,green,blue);"

    for css example: 
                .red{
                color: rgb(255, 0, 0);
            }
            
            .green{
                color: rgb(0, 255, 0);
            }
            
            .blue{
                color: rgb(0, 0, 255);
            }
            
            .black{
                color: rgb(0, 0, 0);
            }
            
            .yellow{
                color: rgb(255, 255, 0);
            }
  3. Hexadecimal of Hex color: for example → color:#ffffff;

    4. Letter:1,2,3,4,5,6,7,8,9,A,B,C,D,E,F(hex code is another form of rgb color)

    Red

    Green

    Blue

    Black

    Yellow

    #ffa511

    note:here in the hex code (#000000) the first two letter repersent value of R(red) then 3rd and 4th letter repersent value of G(Green) then last two letter is repersents value of B(blue).

    for css example: 
                .red1{
                color: #ff0000;
            }
            
            .green1{
                color: #00ff00;
            }
            
            .blue1{
                color: #0000ff;
            }
            
            .black1{
                color: #000000;
            }
            
            .yellow1{
                color: #ffff00;
            }
            
            .another{
                color: #ffa511;
            }
