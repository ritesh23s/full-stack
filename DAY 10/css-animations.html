☆ CSS Animations ☆

⇛ Used for to animate CSS elements.
syntex ⇒

        @keyframes myName{
            from{font-size: 20px;}
            to{font-size: 40px;}
        }

        Where myName = Animation Name (write anything like - abc, hjhj, etc.)

For sets Animation we have to firstlly define following Properties ⇛

  1. animation-name
  2. animation-duration
  3. animation-timing-function
  4. animation-delay
  5. animation-iteration-count
  6. animation-direction
We already read the some properties(animation-duration, animation-timing-function, animation-delay) in transition property
Now we need to read some properties.

Topic:01 ⇛ animation-name.

⇛Define animation name which is we create Like ⇒

        @keyframes colorAnimation{
            from{
                background-color: green;
            }
        
            to{
                background-color: blue;
            }
        }

        Where,
        colorAnimation = Animation Name.
        and

        from{       to{

        }           }

        is Created Animation.

Topic:05 ⇛ animation-iteration-count.

⇛ Its tell how many times show animation at your elements.
Write Like This ⇒

  1. animation-iteration-count: 1/2/3.....;
  2. animation-iteration-count: infinite;
Sets Like This ⇛ 
        .box{
            height: 100px;
            width: 300px;
            background-color: pink;
            animation-name: colorAnimation;
            animation-duration: 5s;
            animation-timing-function: ease-in;
            animation-delay: 0s;
            animation-iteration-count: infinite; // animation show infinite times.
        }

Topic:06 ⇛ animation-direction.

⇛ Its have some specific values for set animation-direction.

  1. normal
  2. reverse
  3. alternate
  4. alternate-reverse
  5. initial
for more animation-direction properties please visit Click Here ..!!
Sets Like This ⇒ 
        .box{
            animation-direction: normal;
            animation-direction: reverse;
            animation-direction: alternate;
            animation-direction: alternate-reverse;
            animation-direction: initial;
        }

Topic: ⇛ Animation Shorthand

Here we can also sets animation in shorthand Like This ⇛ 
        .box{
            height: 100px;
            width: 300px;
            background-color: pink;
            animation: fontAnimation 3s ease-in 0s infinite normal;
        }
        
        Where, 
        fontAnimation = animation-name
        3s = animation-duration
        ease-in = timing-function
        0s = animation-delay
        infinite = animation-iteration-count
        normal = animation-direction

Use annimation Like This ⇒ 
            
            Here we Create animation template (animation blue print) and defined "animation name" 
 
            like - fontAnimation, widthAnimation, colorAnimation
            
            @keyframes fontAnimation{
                from{
                    font-size: 20px;
                }
                
                to{
                    font-size: 40px;

                }
            }

                    width: 10px;

                }
            }
            
            @keyframes colorAnimation{
                from{
                    background-color: green;
                }
                
                to{
                    background-color: blue;
                }
            }

            Now.! here we used multiple animation
            
            and we defned animation properties of the element (.box)             
            .box{
                height: 100px;
                width: 300px;
                background-color: pink;

                animation-delay: 0s;
                animation-iteration-count: infinite;
                animation-direction: normal;
            }
