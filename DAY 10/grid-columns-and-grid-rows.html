☆ Grid Columns ☆

⇛ Defines a particular item's starting and ending position. inside the column.
it's have two properties -

  1. grid-column-start: Line_Number;
  2. grid-column-end: Line_Number;

Explaination Topic: ⇛ Grid-columns.

Sets Like This ⇒ 
            .gridcolumns{
                height: 300px;
                width: 600px;
                background-color: rgb(212, 212, 20);
                display: grid;
                grid-template-rows: repeat(5, 1fr);
                grid-template-columns: repeat(3, 1fr);
            }
            Here nwe sets the grid properties for container

            Now ⇛ we give a particular item's starting & ending position inside the column. Like This ⇒
            
            .one{
                grid-column-start: 1;
                grid-column-end: 3;
            }
            Where, 1 & 3 = Line_number

            We can also write in shorthand Like This ⇛
            "grid-column: start_col/end_col"
            .one{
                grid-column: 1/3;
            }

            Where, 1 & 3 = Line_number
