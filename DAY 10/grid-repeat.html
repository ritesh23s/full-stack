☆ Grid Template Repeat ☆

⇛ Repeat is used to divide all available space in equal (=) size of rows & columns

  1. grid-template-rows: repeat(count, 1fr)
  2. grid-template-columns: repeat(count, 1fr)
    3. 1fr means ⇒ 1 fraction of 100% available space(rows & colomns).
    here count = number of rows & columns which we want to create.

Explaination Topic: Grid Template Repeat.

Sets like this 
            .container{
                width: 600px;
                height: 300px;
                background-color: rgb(217,217,20);
                display: grid;
                grid-template-rows: repeat(5,1fr);
                grid-template-columns: repeat(3, 1fr);
            }
            After sets this properties repeat divide available space in equally.
Note: Prefer always used "grid-template-rows" & "grid-template-columns" both together.
