Explaination Topic:01 ⇛ grid-template-rows: ....;

Its used to create & sets grid rows properties(rows height). Like This ⇒ 
            .gridtemprows{
                width: 600px;
                height: 300px;
                background-color: rgb(217, 217, 20);
                display: grid;
                grid-template-rows: 50px 50px 50px 50px 50px;
            }
            Here we sets like this "50px 50px 50px 50px 50px" all rows height size.

Explaination Topic:02 ⇛ grid-template-columns: ...;

It's used to create & sets grid columns properties(columns width). Like This ⇒ 
            .gridtempcolumns{
                width: 600px;
                height: 300px;
                background-color: rgb(217, 217, 20);
                display: grid;
                grid-template-columns: 20px 20px 20px 20px 20px;
            }

            Here we sets like this "100px 100px 100px" all columns width size.

Explaination Topic: ⇛ grid-template-columns & rows uses.

Here explaination how we can sets grid-template-columns & rows. 
            .gridtempuses{
                width: 600px;
                height: 300px;
                background-color: rgb(217, 217, 20);
                display: grid;
                grid-template-columns: 100px 100px 100px;
                grid-template-rows: 50px 50px 50px 50px 50px;
            }

            here we defined grid-template-columns & rows properties.
            if i want to create an another item inside container then thoese
            item automatically sets in next cell.

            Here another way to sets grid-template Like this ⇒
            .gridtempuses{
                grid-template-columns: auto auto auto;
                grid-template-rows: auto auto auto auto;
            }

            Generally we can't prefer auto
