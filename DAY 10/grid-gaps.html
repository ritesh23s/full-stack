Explaination Topic ⇛ "row-gap" & "column-gap"

Sets Like This ⇒ 
            .container{
                height: 300px;
                width: 600px;
                background-color: rgb(212, 212, 20);
                display: grid;
                grid-template-rows: repeat(5, 1fr);
                grid-template-columns: repeat(3, 1fr);
                row-gap: 10px;      //sets gaps between rows.
                column-gap: 30px;       // sets gaps between columns.
            }

            Here we easily sets the gaps between lines.

            Here shorthand to write gap ⇒
            
            01.grid-gap: 10px 50px;
                Here, 10px = row-gap, & 50px = column-gap
            02.grid-gap: 20px;     
                Here, 20px is both gap row & column.
