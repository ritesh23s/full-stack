☆ Percentage In CSS ☆

⇛ With the help of Percantage in CSS we can sets our animation in stages. Like ⇒ 0%, 10%, 30%, 50%, 80%, 100%, etc.

This is Box

Explain Topic: ⇛ Percentage in CSS

Sets Like This ⇒ 
            .box{
                height: 100px;
                width: 300px;
                background-color: purple;
                border: 2px solid black;
                border-radius: 15px;
                text-align: center;
                margin: 0 auto;
                animation: animate 4s ease-in-out 0.5s infinite alternate-reverse;
            }

            Here we sets annimation in shorthand.
            
            @keyframes animate{
                0%{
                    background-color: red;
                    box-shadow: 2px 2px 10px blue;
                }
            
                50%{
                    background-color: yellow;
                    box-shadow: 2px 2px 10px red;
                }
            
                80%{
                    background-color: blue;
                    box-shadow: 2px 2px 10px purple;
                }
                
                100%{
                    background-color: aqua;
                    box-shadow: 2px 2px 10px black;
                }
            }

            Here, We create template (blueprint) of animation.
