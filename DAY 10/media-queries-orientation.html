Explaination Topic:01 ⇛ Orientation = Landscape

Sets for our screen orientation = landscape 
            Sets Like This ⇛
            @media (orientation: landscape) {
                .box{
                    background-color: skyblue;
                    box-shadow: 2px 2px 8px black;
                }
            
                .demo1{
                    color: green;
                    font-weight: bold;
                }
            }
       
            After sets this media-queries @media (orientation: landscape) properties when our screen orientation = Landscape
            then .box background-color = skyblue and .demo1 color = green, font-weight = bold.
