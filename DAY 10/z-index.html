☆ Z-index ☆

⇛ It decides the stach level of elements.

⇛ Overlapping elements whith a smaller one.

        Syntex ⇒
        
        div {
            z-index: auto (0);
            z-index: 1/2/3....;
            z-index: -1/-2/-3/...;
        }
Note :
  1. By default all elements "z-index" value always auto or 0.
  2. For giving z-index property firstly we have to set thoese element position Like ⇒ Relative, Absolute, fixed, sticky, etc.
  3. We can't sets z-index properties thoese element position is default and static

Explaination Topic:01 ⇛ Z-index 

            .one{
                background-color: red;
                position: relative;
                z-index: 1;
            }
            
            .two {
                background-color: blue;
                position: relative;
                bottom: 60px;
            }

            Here if we sets position bottom: 60px; of ".two" then ".two" element Overlapping
            and cover the ".one" element some part. but 
            If we sets ".one" element position position: relative , fixed, etc and sets z-index: 1; then
            ".one" element Overlapping the ".two" element.
            

                The element have large z-index which is always Overlapping.
