☆ Media Queries ☆

⇛ Its help to create responsive website.

⇛ media-queries have particular features - width(of viewport)

here are multiple media-queries features click here ..!
The most usable media-queries features ⇛ width 
        syntex ⇒

        @media (width: 400px) {
            div{
                background-color: red;
            }
        
            h3{
                color: blue;
                font-weight: bold;
            }
        }

Demo of @media-queries(@media (width: 400px)) which is apply at the exactly a single point.

Explaination Topic:01 ⇛ Demo of @media-queries which is apply at the exactly a single point.

Sets Like This ⇒ 
            @media (width: 400px) {
                .box{
                    background-color: red;
                }
            
                .demo{
                    color: blue;
                    font-weight: bold;
                }
            }

            After sets these media-queries properties (@media (width: 400px))
            when device width touch exactly 400px then div background-color = red, and h3 color = blue and fontp-weight = bold.

Demo of @media-queries (@media (min-width: 500px)) which is apply at the min width & above than min-width.

This is a Box1

Explaination Topic:02 ⇛ (@media (min-width: ....;)) which is apply at the min width & above than min-width

Sets Like This ⇒ 
            @media (min-width: 500px) {
                .box1{
                    background-color: yellowgreen;
                }
            
                .demo1{
                    color: orangered;
                    font-weight: bold;
                }
            }

            After sets this media-queries properties (@media (min-width: 500px)) 
            when the device screen width touch 500px and more than 500px
            then .box1 background-color = yellowgreen and .demo1 color = orangered , font-weight = bold.

Explaination Topic:04 ⇛ @media (min-width: ....;) and (max-width: ....;)

Whith the help of and (min-width: 200px) and (max-width: 300px) we can sets our range between given dimention of width. 
            Sets Like This ⇒
            @media (min-width: 200px) and (max-width: 300px) {
                .box3{
                    background-color: springgreen;
                }
            
                .demo3{
                    color: brown;
                }
            }


            After sets this media-queries properties @media (min-width: 200px) and (max-width: 300px)
            when device width touch 200px to 300px width then .box3 background-color = springgreen;
            and .demo3 color = brown.
