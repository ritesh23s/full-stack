☆ Common Properties of Grid. Which is used for element's alignment ☆

For Horizontal Alignment.

  1. justify-items: ...; (used for container)
  2. justify-self: ...; used for item

For Vertical Alignment.

  1. align-items: ...; (used for container)
  2. align-self: ...; used for item

Place-Items

  1. place-items: ...; (used for container Horizontal & Vertical Both)
  2. place-self: ...; used for item

Explaination Topic: ⇛ Place.

Sets Like This ⇛ 
            .gridplace{
                height: 300px;
                width: 600px;
                background-color: rgb(212, 212, 20);
                display: grid;
                grid-template-rows: repeat(3, 1fr);
                grid-template-columns: repeat(3, 1fr);
                grid-gap: 10px;
                place-items: center;
            }
            Here, If we sets place-items: center; then all items place in center inside overall container's cell.

            Now -
            .one{
                place-self: start;
            }

            If we sets place-self: start; then a particular item (.one) position sets inside a cell at the start.
Note: "place sets both alignments "Horizontal & Vertical""
