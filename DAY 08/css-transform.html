☆ CSS Transform ☆

⇛ This property lets you "rotate, scale, skew, or translate" an element

Explaination Topic:01 ⇛ Rotate

With the help of "transform-rotate: ....;" we can rotate an element. Like This ⇒ 
            #one{
                transform: rotate(45deg);
                transform: rotate(180deg);
                transform: rotate(-45deg);
            }

Explaination Topic:02 ⇛ Scale

With the help of "transform: scale(....);" we can scale an element. Like This ⇒ 
            #two{
                transform: scale(1.5);
            }

            Here if we want to sets scale along to diffrent direction (X & Y) then ⇒
            #two{
                transform: scale(0.5,1.5);
            }

            Here if we want to sets scale only one direction ⇒
            #two{
                transform: scaleX(2);
            }
            
            #two{
                transform: scaleY(1.5);
            }
            
            #two{
                transform: scaleZ(3);
            }
Note: Element scale both (Y - direction & X -dicrection)

Explain Topic:03 ⇛ Translate

With the help of "transform: translate (....);" we can translate(move along to X & Y direction) an element. Like This ⇒ 
            #three{
                transform: translate(100px, 100px);
            }

            Here if we want to sets translate along to only one direction Then ⇒
            #three{
                transform: translateX(50px);
            }

            #three{
                transform: translateY(30px);
            }
Note: Translate Movement along width only 2D.

Explaination Topic:04 ⇛ Skew

With the help of "transform: skew(....);" we can skew an element. Like This ⇒ 
            #four{
                transform: skew(30deg);
                transform: skew(-30deg);
            }

            Here we can also set skew for X & Y direction
            #four{
                transform: skewX(50deg);
            }
            
            #four{
                transform: skewY(35deg);
            }

            Here we can sets multiple transformation for an element Like This ⇒
            #five{
                transform: rotate(20deg) translateX(10px) skew(12deg) scaleY(1.05);
            }

☆ Box Shadow ☆

Explaination Topic:01 ⇛ Box Shadow

With the help of box shadow we can adds "shadow" effects around an element's frame. Like This ⇒ We can also give 3D effect an element with the help of "box-shadow" 
            .boxshadow{
                height: 200px;
                width: 200px;
                background-color: red;
                box-shadow: 2px 2.5px 10px black;
            }
            Where,
            2px = X-ofset (distance along with x-axis)
            2.5px = Y-ofeset (distance along with y-axis)
            10px = blur radius of the shadow (value of blurness of the shadow)
