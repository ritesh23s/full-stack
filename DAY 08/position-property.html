☆ Position Property ☆

⇛ The position CSS property sets how an element is positioned in a document
The top, right, bottom, and left properties determine the final location of positioned elements.

⇛ in another word we can sets the any particular element position in our overall document.

  1. Static
  2. Relative
  3. Absolute
  4. Fixed

Explaination Topic:01 ⇛ Positoin : Static

The top, right, bottom, left, and z-index properties have no effect.
This is the default value. Sets the position Like This ⇒ 
            .static{
                background-color: yellow;
                position: static; 
                top: 500px; // determine final position along with "top"
            }

            Here we determine the final position "top: 500px;" 

            but here we can't see any effect because of "position: static;" 

            and static position is default.
Note: static position is default position.

Explaination Topic:02 ⇛ Positoin : Relative

The offset(determined position like - top: 100px;, left: 200px; etc.) is relative to itself based on the values of top, right, bottom, and left. Like This ⇒ 
            .relative{
                background-color: blue;
                position: relative;
                top: 60px; //determined final position along with itself "top: 60px;"
                left: 15px; //determined final position along with itself "left: 15px;"
            }
Note: The relative position Positionate along with itself.

Explaination Topic:03 ⇛ Position: Absolute

The element is removed from the normal document flow, and no space is created for the element in the page layout.
It is positioned relative to its closet positioned ancestor, if amy; athorwise, it is placed relative to the initial containing block. Like This ⇒ 
            .absolute{
                background-color: red;
                position: absolute;
                top: 150px;
                left: 200px;
            }
Note:
  1. if we don't sets any position (except "static position") of the parent element (section) then the child element is positioned along with overall document.
  2. if we sets the position of parent element then child element is positioned along with their parent element.
  3. Conclusion: In absolute position element positioned along with their closer element which have already given position except "static position"
  4. Here absolute say that for positioned element which have position: absolute; not created extra space

Explaination Topic:04 ⇛ Position: Fixed

The element is removed from the normal document flow, and no space is created for the element in the page layout.
It is positioned relative to the initial block established by the viewport. Like This ⇒ 
            .fixed{
                background-color: greenyellow;
                position: fixed;
                top: 100px;
                right: 200px;
            }
