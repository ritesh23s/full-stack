Explaination Topic:01 ⇛ Alpha Channel

Sets the opacity for a color. Range - 0 to 1
example - rgba(255,255,255,0.3) 
            .alphachannel{
                height: 200px;
                width: 200px;
                background-color: rgba(255, 0, 0, 0.5);
            }

            here we sets alpha value = "0.5"

Explaination Topic:02 ⇛ Opacity

Sets the opacity for the element. Range - 0 to 1
Example - opacity: 0.5 
            .opacity{
                height: 200px;
                width: 200px;
                background-color: #ffa511;
                opacity: 0.5;
            }

            Here we sets opacity for the element = "opacity: 0.5;"
