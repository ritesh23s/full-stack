Explaination Topic:01 ⇛ Height

By default it sets the "content area height" of the element. Like This ⇒ 
      .box-model{
        height: 200px;
      }
Note: Height always set only for "content" not for overall elemtnt.

Explaination Topic:02 ⇛ Width

By default, it sets the "content area width" of the element. Like This ⇒ 
      .box-model{
        width: 600px;
      }

Explaination Topic:03 ⇛ Border

Used to set an element's border. There are diffrent type of border properties.
  1. border-width
  2. border-style
  3. border-color
Explaination ⇒ 
      .box-model{
        border-width: 5px;
        border-style: solid;
        border-color: blue;
      }

      
      
Explaination Subtopic: ⇛ Border Shorthand 

      ⇛ Here we can also give the all properties in one line. Like this → 
      
      .box-model{
        border: 5px solid blue;
      }  
      
      
      
      
    
Explaination Subtopic:01 ⇛ Border width
  
    ⇛ There are advance type of sets "border-width" Like This →
   
    .box-model{
      border-style: solid;
      border-top-width: 5px;
      border-left-width: 10px;
      border-right-width: 15px;
      border-bottom-width: 20px;
    }
    

    
    
Explaination Subtopic:02 ⇛ Border Style 

    ⇛ There are different typs of "border-style" Like This → 
    
    h1{
      border-width: 2px;
      border-style: solid;
      border-style: dashed;
      border-style: dotted;
      border-style: double;
      border-style: groove;
      border-style: hidden;
      border-color: red;
    }
    

    ⇛ There are advance type of sets "border-style" Like This → 

    h1{
      border-width: 5px;
      border-color: red;
      border-left-style: dashed;
      border-right-style: double;
      border-top-style:dotted;
      border-bottom-style: groove;
    }
    
    
    
Explaination Subtopic:03 ⇛ Border Color 

    ⇛ There are advance type of sets "border-color" Like This → 
    
    .box-model{
      border-style: solid;
      border-top-color: red;
      border-right-color: orangered;
      border-left-color: yellow;
      border-bottom-color: yellowgreen;
    }

Explaination SubTopic: of Border: ⇛ Border Radius

Used to "round the corners" of an element's outer border edges. Like This ⇒ 
    button{
        border: 2px solid red;
        border-radius: 10px;
    }


    ⇛ There are advance type of sets "border-radius" Like This → 
    .box-model{
      border-top-left-radius: 15px;
      border-top-right-radius: 20px;
      border-bottom-right-radius: 35px;
      border-bottom-left-radius: 45px;
    }

Explaination Topic:04 ⇛ Padding

Sets the padding of an element. Like This ⇒ 
    .box-model{
      padding: 50px;
    }


    ⇛ There are advance(padding sides) type of sets "padding" Like This → 
    
    .box-model{
      padding-top: 20px;
      padding-bottom: 15px;
      padding-left: 25px;
      padding-right: 30px;
    }

    button{
      background-color: yellow;
      padding-left: 25px;
      padding-right: 25px;
    }

    
    
Explaination Subtopic: ⇛ Padding Shorthand

    #1. for all 4 sides ⇒
    
    .box-model{
      padding: 50px;
    }
    

    #2. for top & bottom | left & right ⇒
    
    .box-model{
      padding: 1px 2px;
    }
    
    #3. for top|left & right|bottom ⇒
    
    .box-model{
      padding: 1px 2px 3px;
    }
    
    #4. for top|right|bottom|left  (Clockwise) ⇒
    
    .box-model{
      padding: 1px 2px 3px 4px;
    }

Explaination Topic:05 ⇛ Margin

To set spacing on the "outside" of border. Like This ⇒ 
    .box-model{
      margin: 50px;
    }

    ⇛ There are advance(margin sides) type of sets "margin" Like This → 
    
    .box-model{
      margin-left: 5px;
      margin-right: 10px;
      margin-top: 20px;
      margin-bottom: 25px;
    }
