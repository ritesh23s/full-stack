Explaination Topic:01 ⇛ Display inline

Takes only the space required by the element. no linebreaks before and affter setting width/height and margin/padding (only top & bottom margin/padding) are not allowed. Sets Like This ⇒ 
            #display{
                height: 200px;
                width: 200px;
                background-color: greenyellow;
                border: 2px solid black;
                margin: 5px;
                display: inline;
            }
Note:
  1. display:inline; property say that we don't need new line space around my "top and buttom" and any element easyly stay in my any side.
  2. Here's if we give display:inline; then height & width has not effect on this element
  3. If we give display:block; then height & width effect on this element
  4. Important - Display:inline; properties always take only required space by element

Explaination Topic:02 ⇛ Display Block

Takes full space available in width and leaves a newline before and affter the element. Sets Like This ⇒ 
            #display{
                height: 200px;
                width: 200px;
                background-color: greenyellow;
                border: 2px solid black;
                margin: 5px;
                display: block;
            }
Note:
  1. We can be easyly set width & height , margin & padding with display:block; properties
  2. display:block; property say that we need new line space around my "top and buttom" and i don't want to any element stay in my any side.
  3. Important - Display:block; properties always take full space by element

Explaination Topic:03 ⇛ Display Inline Block

It is similar to display:inline; but setting height,width,margin & padding is allowed element can seet each other 
            If we want to set any element position in inline and define
            width & height , margin & padding then we always use display:inline-block properties. Like This ⇒

            #display{
                height: 200px;
                width: 200px;
                background-color: greenyellow;
                border: 2px solid black;
                margin: 5px;
                display: inline-block;
            }

            Here's we have set width & height , margin & padding in #display with the help of display:inline-block; properties
Note:
  1. We can be easyly set width & height , margin & padding with display:inline-block; properties
