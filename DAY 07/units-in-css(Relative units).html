Explaination Topic:01 ⇛ Percentages (%)

It is often used to define a "size as relative to an element" object. Example: ⇒ 
                width: 33.33% // relative to the parent.
                margin-left: 50% // relative to the parent size.
            
            Here we want to sets "H1" width according to his parent (< body>) so we sets like this →
            h2{
                border: 2px solid black;
                width: 33.33%;
            }

            Here another example with the help of "margin property" Like ⇛
            Here we sets the ".outer" margin: 50px;
            .outer{
                height: 400px;
                width: 400px;
                background-color: pink;
                margin: 50px;
            }

            now we want to sets the "margin" of ".inner" according to his parent ".outer" size so we can sets like this ⇒
            .inner{
                height: 100px;
                width: 100px;
                background-color: cornflowerblue;
                margin-left: 10%;
            }

Explaination Topic:02 ⇛ Em

Unit Relative to
em Font size of the parent, in the case of typographical properties like: "font-size", and font size of the element itsels, in the case of other properties like "width"
⇛ "em" used to sets relative to parent size(text-related properties or another properties). 

            Example - 01

            Here if we sets "font-size" of the parent div (.outerem) then →
            .outerem{
                height: 400px;
                width: 400px;
                border: 2px solid black;
                background-color: pink;
                font-size: 20px;
            }

            Now, here we want to sets "font-size" of ".innerem" relative (according) to parent (.outerem) then →

            .innerem{
                height: 100px;
                width: 100px;
                background-color: cornflowerblue;
                font-size: 1.5em;
            }

            here ".innerem" font-size sets relative (according) to parent(.outerem) font-size(20px), Here final font-size of (.innerem) = 20px * 1.5em = 30px.

            Example - 02

            Here we can controle our padding according to "font-size" Like This ⇒

            button{
                font-size: 10px;
                border: 5px solid green;
                border-radius: 15px;
                /* padding: 0 20px 0 20px; */
                padding: 0 1em 0 1em;
            }

            Here we sets padding-left & padding-right = 1em, Then now,
            if font-size increase then padding automatically increase
            according to "font-size".

Subtopic: ⇛ Drawback of em

Snowball effect in nested elements 
            Example - 

            section{
                font-size: 15px;
            }

            #nesteddiv{
                font-size: 1.5em;
            }

            Here if we sets "font-size" of #nesteddiv = 1.5em then only "div1" changed our font size according to "section"
            otherwise all #nesteddiv changed our font size according(relative) to each other. Like → div1 < div2 < div3
            Means #nesteddiv note changed our size according to "section"
Note: for avoide em drawback we need to another unite that called "rem" (Root em)

Explaination Topic:03 ⇛ Rem (Root Em)

Unit Relative to
rem Font size of root element
⇛ "rem" used to sets size according to root element (in nested elements) 
            Example -

            section{
                font-size: 15px;
            }
            
            #nesteddiv{
                font-size: 1.5rem;
            }

            Here if we sets "font-size" of #nesteddiv = 1.5rem then all divs sets our
            "font-size" according(Relative) to root element (section) "font-size".
