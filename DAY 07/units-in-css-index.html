Size, Position & List

There are more unts for discribing size other than "px"
There are following:- rem, em, vw, vh, percentages etc.

What's wrong with pixels (px) ..?

Pixles are relative to the viewing device.
For a device with size 1920 * 1080, 1px is one unit out of 1080/1920.

Relative Lengths

These units are relative to the other length property.
Following are some of the most commonly used relative lengths:-

List-style property

The list style property is a shorthand for type, position & image.

                ul{
                    list-style: square inside url(/img/cpu.jpg);
                }

                Where: 
                    square → List-Style-Type
                    Inside → List-Style-Position 
                    cpu.jpg → List-Style-image

Z - index property

z-index property specifies the stack order of an element.

It define which layer will be above which in case overlapping elements.
