The Display Property

The CSS display property is use to determine whether an element is treated as a block/inline & the layout used for children Children → Flexbox/Grid/etc.

Display:inline;

Takes only the space required by the element. no linebreaks before and affter setting width/height and margin/padding (only top & bottom margin/padding) are not allowed.

Note - We can't set width & height , margin & padding with display:inline; properties 
Here's two boxes {box1(bg-color= red) & box2(bg-color= blue)}
BOX1
BOX2 
                Important - Display:inline; properties always take only required space by element 


                .box1{
                    background-color: red;
                    /* width: 150px; */
                    /* height: 300px; */
                    display: inline;
                }
                
                .box2{
                    background-color: blue;
                    /* width: 150px; */
                    /* height: 300px; */
                    display: inline;
                }
                Here's if we give display:inline; then height & width has not effect on this element 

                
                    display:inline; property say that we don't need new line space around my "top and buttom" and any element easyly stay in my any side.

                note - If we give display:block; then height & width effect on this element

Display:Block;

Takes full space available in width and leaves a newline before and affter the element.

Note - We can be easyly set width & height , margin & padding with display:block; properties 
Here's two boxes {block1(bg-color= red) & block2(bg-color= blue)}
BLOCK1
BLOCK2 
                Important - Display:block; properties always take full space by element 
                
                .block1 {
                    background-color: red;
                    width: 100px;
                    display: block;
                }

                display:block; property say that we need new line space around my "top and buttom" and i don't want to any element stay in my any side.
                
                .block2 {
                    background-color: blue;
                    display: inline;
                }
                Here's display:inline; that means here only take required space by the element and we can't set width & height.

Display:inline-block;

It is similar to display:inline; but setting height,width,margin & padding is allowed element can seet each other

Note - We can be easyly set width & height , margin & padding with display:inline-block; properties 
Here's two boxes {inlineblock1(bg-color= red) & inlineblock2(bg-color= blue)}
INLINEBLOCK1
INLINEBLOCK2 

            

            If we want to set any element position in inline and define
            width & height , margin & padding then we always use display:inline-block properties
            

                .inlineblock1{
                    background-color: red;
                    width: 200px;
                    height: 50px;
                    margin: 10px;
                    padding: 5px;
                    display: inline-block;
                    }

                    Here's we have set width & height , margin & padding in 
                    INLINEBLOCK1 Box with the help of display:inline-block; properties
                    
                    
                    .inlineblock2{
                        background-color: blue;
                    display: inline;
                    }
                
                Here's we give only display:inline; properties for seet INLINEBLOCK1 to INLINEBLOCK2 side, here we can't define 
                width & height, margin & padding beacouse of display:inline; properties

Display:none; vs Visiblity:hidden;

With display:none; the element is removed from the document flow Its space is not blocked.

With Visiblity:hidden; the element is hidden but its space is reserved.

Here's two boxes {DISPLAY:NONE;(bg-color= red) & VISIBILITY:NONE;(bg-color= blue)}
DISPLAY:NONE;
VISIBILITY:NONE; 

                .displaynone{
                    background-color: red;
                    width: 200px;
                    height: 50px;
                    margin: 5px 0px;
                    display: none;
                }

                Here's if we give displa:none; then element is removed from document flow and element space is not blocked

                .visibilityhidden {
                    background-color: blue;
                    width: 200px;
                    height: 50px;
                    visibility: hidden;
                }

                Here's if we give visibility:hidden; then element element is hidden but its space is reserved (Its space is blocked.)
