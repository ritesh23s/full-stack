☆ Cascading Stylesheet ☆

⇛ The "CSS cascade algorithm"'s job is to select CSS declaration in order to determine the correct values for CSS properties

Explaination: Cascading - Stylesheet

Cascading StyleSheet

Cascading stylesheet always apply on same elements. Like This ⇒ 
            h3{
                background-color: yellow;
            }
            
            h3{
                background-color: blue;
            }


            Here we gate multiple css property value in "h3" (background-color: yellow; & background-color: blue;)but css value
            apply on "h3" which is we give in last time (background-color: blue;) 
            
            h3{
                background-color: blue;
            }
Note:
  1. Cascading stylesheet always apply on same elements.
  2. Cascading stylesheet always apply on thoese element which we give property last time.
  3. Cascading stylesheet always apply css including file like
    < link rel="stylesheet" href="css/cascading-stylesheet.css">
    < link rel="stylesheet" href="css/cascading-stylesheet2.css">
