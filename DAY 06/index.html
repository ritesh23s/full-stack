Explaination Topic:01 ⇛ :hover pseudo class.

syntax of ":hover" 
                button:hover{
                    background-color: black;
                    color: white;
                }
                
                after that when we go to the our button by mouse cursor then button "background-color" is getting black & "color" is getting white.
                
                Here same with H3 
                h3:hover{
                    text-decoration: none;
                    text-decoration: overline;
                }

Explaination Topic:02 ⇛ :active pseudo class.

Whenever we clicked the any element then those element get active state. then we getting effect that element with the help of ":active pseudo class". Like This ⇛ 
                button:active{
                    background-color: hotpink;
                }

                Here if we clicked the button where used "button:active" after that button "background-color" getting hotpink. 
                
                Here same with "p"
                p:active{
                    font-weight: bold;
                    color: green;
                }
