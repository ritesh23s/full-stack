What is Bootstrap .?

⇛ Bootstrap is a powerful, feature-packed frontend toolkit, Build anything-from prototype to production-in minutes.

⇛ Bootstrap is a special framework which is work on flex-box concepts.

⇛ It's have many prebuild components (css files) Like ⇒ botton, forms, navbar, etc.

Benefits of Bootstrap ⇛

⇛ Makes Development Faster & Easiar.

⇛ Good choice of pre-build Components (Which is already responsive).

⇛ Good Browser Compatibility.

How to include Bootstrap .?

  1. By adding CDN Links in your HTML Document ⇒

    2. CDN stand for Contemnt Delivery Network

                For CSS ⇒ https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css
            For JavaScript ⇒ https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js

    Find This Link ⇛

                go to getbootstrap.com < click Read the doc < introduction < CDN Links Section

    How include in HTML Document ⇛

                For CSS ⇒ < link rel="stylesheet" href="" >
            Like ⇒     
            < link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css">     

            For JavaScript ⇒ inside < body> then  << script src=""> < / script>
            Like ⇒ 
            << script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js">< /script >
