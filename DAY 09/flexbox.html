☆ Flexbox Direction ☆

⇛ It sets how flex items are placed in the flex container, along which axis and direction. Like This ⇒

  1. flex-direction: row; (main axis, left to right)
  2. flex-direction: row-reverse; (main axis, right to left)
  3. flex-direction: colomn; (cross axis, top to bottom)
  4. flex-direction: column-reverse; (cross axis, bottom to top)
Note:
  1. "flex-direction" is by default sets "row" always.
  2. flex-deretion ignore the given width of child element when given width is overflow. its sets with according to itself(parent element)

Explaination Topic:01 ⇛ flex-direction: row; (main axis, left to right)

With the help of flex-direction: row; we can sets the flex item direction. Like This ⇒ 
            #rowandrowreverse{
                height: 300px;
                width: 80%;
                background-color: azure;
                margin: 10px auto;
                border: 2px solid black;
                display: flex;
                flex-direction: row;
            }
            Here if we can sets "flex-direction: row;" then flex item direction sets "row wise"
Note: flex-deretion: row; is default flex direction

Explaination Topic:02 ⇛ flex-direction: row-reverse; (main axis, right to left)

With the help of "flex-direction: row-reverse;" we can sets the flex item direction. Like This ⇒ 
            #rowandrowreverse{
                height: 300px;
                width: 80%;
                background-color: azure;
                margin: 10px auto;
                border: 2px solid black;
                display: flex;
                flex-direction: row-reverse;
            }
            
            Here if we can sets the "flex-direction: row-reverse;" then flex item direction sets row wise but reverse way.(right to left)

Explaination Topic:03 ⇛ flex-direction: colomn; (cross axis, top to bottom)

With the help of "flex-direction: column;" we can sets the flex item direction. Like This ⇒ 
            #columnandcolumnreverse{
                height: 300px;
                width: 80%;
                background-color: azure;
                margin: 10px;
                border: 2px solid black;
                display: flex;
                flex-direction: column;
            }

            Here is we can sets "flex-direction: column;" then flex items direction sets column wise.

Explaination Topic:04 ⇛ flex-direction: colomn-reverse; (cross axis, bottom to top)

With the help of "flex-direction: column-reverse;" we can sets the flex item direction. Like This ⇒ 
            #columnandcolumnreverse{
                height: 300px;
                width: 80%;
                background-color: azure;
                margin: 10px;
                border: 2px solid black;
                display: flex;
                /* flex-direction: column; */
                flex-direction: column-reverse;
            }

            Here is we can sets "flex-direction: column;" then flex items direction sets column wise but reverse way. (bottom to top)
