☆ Flex Sizing ☆

⇛ Whith the help of Flex Sizing we can controle the extra space inside a flex container (div).
⇛ We can easily Grow & Shrink our flex items with the help of Flex Sizing.

  1. flex-basis: ....;
  2. flex-grow: ....;
  3. flex-shrink: ....;
Note: Its height & width always depends on "flex-direction:.....;" if,
  1. flex-direction: row;, then our main size = width
  2. flex-direction: column;, then our main size = height

Explaination Topic:01 ⇛ flex-basis: ....;

Its sets the individual main size of a flex-item. Like This ⇒ 
            #flexbasis{
                height: 600px;
                width: 80%;
                background-color: azure;
                margin: 10px auto;
                border: 2px solid black;
                display: flex;
                flex-direction: row;
                justify-content: flex-start;
                align-items: center;
            }
            
            Here we can sets the main size (height & width ) of an andividual flex item like this ⇒

            #flexbasisgreen{
                flex-basis: 150px;
            }
            
            #flexbasisorange{
                flex-basis: 200px;
            }

Explaination Topic:02 ⇛ flex-grow: ....;

It spacifies how moch to the flex container's remaining space should be assigned to the flex item's main size. Like This ⇒ 
            #flexgrow{
                height: 600px;
                width: 80%;
                background-color: azure;
                margin: 10px auto;
                border: 2px solid black;
                display: flex;
                flex-direction: row;
                justify-content: flex-start;
                align-items: center;
            }
            After given these property we have some extra spaces inside my container(parent element). 
            if we want to grow our flex item size then we get the property of our individual flex item. Like This ⇒

            #flexgrowvoilet{
                flex-grow: 1;
            }
            Here, if we sets the one individual flex item flex-grow: ...; then this element grow our size(width) and capture the extra space.

            #flexgrowyellow{
                flex-grow: 2;
            }

            Here, if we sets the multiple individual flex item flex-grow: ...; then all flex items grow
            their size (width) and distribute the extra space according to given "flex-grow" value. 

            We can also sets the widht&height Limit. Like This ⇒
            #flexgrowvoilet{
                max-width: 200px;
                min-height: 100px;
            }

            if we sets the size limit (max-width & min-width) these element size always sets according to given value.
Note:
  1. to every individual items grow value is always "flex-grow: 1;"
  2. "flex-grow: 1;" here we used unitless value like - 1, 1.5,2, etc

Explaination Topic:03 ⇛ flex-shrink: ....;

It sets the flex shrink factor of a flex item. Like This ⇒ 
            #flexshrink{
                height: 200px;
                width: 200px;
                background-color: azure;
                margin: 10px auto;
                border: 2px solid black;
                display: flex;
                flex-direction: row;
                justify-content: flex-start;
                align-items: center;   
            }

            After sets these property value we have to see that the flex items(child element) is overflow with their parent element.
            so,
            if we want to decrease our flex items(child element) size of an individual item then use flex-shrink:....; Like This ⇒
            
            #flexshrinkblue{
                flex-shrink: 2;
            }
            
            #fleshrinkgred{
                flex-shrink: 2;
            }

            After that size decrease of these individual item according to given value.
Note: "flex-shrink: 1;" is default value.
