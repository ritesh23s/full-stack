☆ Justify Content ☆

⇛ It's tells how the browser distributes space between and aroundcontenet items along with the main-axis.

  1. justify-content: flex-start;
  2. justify-content: flex-end;
  3. justify-content: center;
  4. justify-content: space-between;(space-around, space-evenly)
Note: justify-content always depends on flex-direction Like ⇒
  1. flex-deretion: row, row-revers;
  2. flex-deretion: column, column-revers;

Explaination Topic:01 ⇛ justify-content: flex-start;

With the help of "justify-content: flex-start;" we can sets space between "flex items" Like This ⇒ 
            #flexstart{
                height: 300px;
                width: 80%;
                background-color: azure;
                margin: 10px auto;
                border: 2px solid black;
                display: flex;    
                justify-content: flex-start ;
            }

            Here if we sets "jtjustify-content: flex-start;" then all flex items go to main-axis start.

Explaination Topic:02 ⇛ justify-content: flex-end;

With the help of "justify-content: flex-end;" we can sets space between "flex items" Like This ⇒ 
            #flexend{
                height: 300px;
                width: 80%;
                background-color: azure;
                margin: 10px auto;
                border: 2px solid black;
                display: flex;    
                justify-content: flex-end;
            }

            Here if we sets " justify-content: flex-end" then all flex items arranged at the end of main-axis.

Explaination Topic:03 ⇛ justify-content: center;

With the help of "justify-content: center;" we can sets space between "flex items" Like This ⇒ 
            #center{
                height: 300px;
                width: 80%;
                background-color: azure;
                margin: 10px auto;
                border: 2px solid black;
                display: flex;    
                justify-content: center;
            }

            Here if we sets " justify-content: center" then all flex items arrange at the center of main-axis

Explaination Topic:04 ⇛ justify-content: space-between;(space-around, spae-evenly)

With the help of "justify-content: space-between;(space-around, space-evenly)" we can sets space between "flex items" Like This ⇒ 
            #spacebetwwenaroundevenly{
